# RagnaBot

A multi featured Ragnarok Discord bot for Arcadia.

# Commands

```
/mvp <mvp-key> 11:26
/mvp <mvp-key> 1126
/mvp <mvp-key> 5m
/market add <item-id> <maximum-price>
/market remove <item-id>
/market list
/timetag timezone iso_datetime
```
